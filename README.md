# Mini-signal-generator
Mini signal generator, capable of operating as a function generator. The module generates square, sawtooth, triangular, and sinusoidal waveforms. The NE555 circuit generates a square wave with a duty cycle of approximately 50%. The frequency of the generated waveform depends on the values of the selected components. This signal is then converted in subsequent stages into signals of other shapes. All these signals are fed to the inputs of the circuit, which is an analog multiplexer. Depending on the logic level at the control inputs A and B of the integrated circuit (and the signal selector jumper circuit), it connects one of the four obtained signals to the output. Project designed and created during electronics lessons at university.
